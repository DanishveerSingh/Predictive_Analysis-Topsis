# Title: Topsis

# 1. Methodology

Data Collection ----> Calculating Root Square Sum for each feature ----> Normalizing every attribute ----> Finding ideal value ----> Calculating Euclidean distance ----> Finding Topsis Score ----> Ranking based on topsis score

# 2. Description

- Dataset: data.xlsx
- Weights: Non-negative (> 0)
- Impacts: Positive/Negative (+/-)
- Output: Output.csv

# 3. Objectives

- Applying basic mathematics to fing the topsis score.
- Ranking data points based on topsis score.
- Changing weights and impacts to see difference in score.
- Storing the result in output.csv file.
